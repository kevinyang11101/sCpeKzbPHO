# 图书馆管理系统 python341

## 项目概述

图书馆管理系统是基于Python实现的一个系统，旨在通过数字化手段提高图书馆的运营效率。

## 技术栈

- **前端**：html、css、js
- **开发语言**：Python
- **后端**：Django
- **数据库**：sqlite、MySQL（作者已支持sqlite到MySQL的数据迁移，确保运行无问题）

## 功能模块

### 登录模块

用户可以通过输入用户名和密码来登录系统。未注册用户需要先注册。

### 菜单功能模块

- 查看书籍总数
- 查看会员总数
- 查看书籍类别总数
- 查看出版社总数
- 查看最近归还记录
- 查看重要事件

### 借阅记录模块

管理员可以查询和操作借阅记录的详细信息。

### 书籍管理模块

- 查看所有书籍信息
- 对书籍信息进行增删改查

### 书籍类别管理

- 查看所有书籍类别信息
- 对书籍类别进行增删查操作

### 出版社管理

- 查看出版社信息
- 对出版社信息进行增删改查

### 个人管理模块

用户可以查看和修改个人信息，包括个人资料和借阅历史。

## 系统角色

- 用户：登录、查看和修改个人信息
- 管理员：管理书籍、借阅记录、书籍类别、出版社信息
